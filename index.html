常见问题解答

这个工具是做什么的？

这是一个专为算法学习者打造的牛客网刷题进度追踪器。它提供了每日打卡、进度追踪、全站排名和精准定位等功能，旨在帮助您更高效地在牛客网刷题，养成良好的学习习惯。

核心功能之一是通过直观的颜色系统，帮助您快速识别题目状态，如下图所示：

· 绿色：代表您已通过的题目。
· 橙色：代表您未做、但对比用户已通过的题目。
· 白色：代表双方都未通过的题目。

当您完成某个知识点或某场比赛的所有题目时，知识点/比赛名称也会变为绿色。

三个板块分别适合谁？

竞赛板块： 适合想要提升算法竞赛能力的同学，内容涵盖牛客周赛、小白月赛、各类练习赛及区域赛真题，是锻炼代码思维和解题速度的绝佳平台。

算法学习板块： 适合算法入门者。您可以从"新手入门"系列开始，循序渐进地学习算法与数据结构。此板块还包含“输入输出练习”，可以帮助您快速适应在线笔试中常见的“ACM模式”的输入输出要求。

笔面试板块： 专为准备技术面试的同学设计，汇集了各大互联网公司的校招真题，是面试准备的宝库。

我的个人信息安全吗？

完全安全。本工具所有数据均直接来自牛客网官方公开接口，不经过任何第三方服务器，不存储、不分析、不滥用您的任何个人信息。您可以随时停止使用，我们不会保留任何痕迹。

排行榜是怎么排名的？

我们目前提供两种排行榜，规则如下：

过题排行榜： 根据用户在牛客网“练习”模式下通过的题目总数进行实时排名。
打卡排行榜： 根据用户在“每日一题”功能中的累计打卡天数进行排名。

您可以随时搜索自己的排名，如果排名在1000名之后，系统会为您特别显示。请记住，排行榜是激励进步的工具，专注于自身成长更为重要。

如何找到我的 User ID？

非常简单：登录牛客网后，点击右上角头像进入“我的主页”，浏览器地址栏 `users/` 后面的那串数字，就是您的 User ID。例如，如果地址是 https://www.nowcoder.com/users/12345678，那么您的 User ID 就是 12345678。

每日打卡功能怎么用？

每日打卡是养成刷题习惯的绝佳方式。我们会每天精选一道优质题目，您在完成后可以点击打卡，并通过可视化的打卡日历直观地看到自己的坚持与进步。连续打卡会获得激励，帮助您更好地坚持下去。

新手如何开始？

我们为不同方向的同学推荐了不同的学习路径：

算法竞赛方向： 建议从“练习”板块的“新手入门”开始，结合每日打卡培养手感，然后逐步挑战“竞赛”板块的各类比赛题目，并对练习赛和区域赛真题进行补题训练。

求职面试方向： 同样建议从“新手入门”开始打好基础，并坚持每日打卡。然后，重点转至“校招”板块练习大厂真题，并集中学习“面试高频题”模块。定期回顾，查漏补缺。

请记住，学习算法是一个循序渐进的过程，坚持就是胜利。
