常见问题解答

这个工具是做什么的？

这是一个牛客网刷题进度追踪器。它可以帮助用户方便地查看竞赛题目、追踪自己的完成情况，并与他人进行对比。本工具旨在帮您定位知识盲点，见证自己的成长。

“竞赛”、“校招”、“练习”板块有什么区别？

竞赛 (Contests): 主要收录牛客网定期举办的各类编程竞赛，如周赛、小白月赛、练习赛等。适合希望提升算法竞赛能力、锻炼代码思维和解题速度的用户。
校招 (Campus): 汇集了各大知名企业的校园招聘真题和模拟题。如果你正在准备技术面试和笔试，这个版块将是你的宝库。
练习 (Practice): 包含了从入门到进阶的各类经典算法和数据结构练习题，以及面试高频题。适合系统性学习、巩固基础知识、查漏补缺的用户。

数据来源是哪里？我的账号安全吗？

本工具的数据完全来自于 牛客网 官方接口，并由前端直接向牛客网请求，不经过任何第三方服务器。您的牛客 User ID 仅用于查询您在牛客网上的公开做题记录，工具本身不会存储、分析或泄露您的任何个人信息，请放心使用。

排行榜的数据来源是什么？准确吗？

排行榜的数据同样来自于牛客网官方接口，展示了全站练习题目通过总数最多的前 1000 名用户。您可以通过搜索自己的 User ID 来查询您在榜单中的具体位置。如果您的排名在 1000 名以后，系统也会将您的排名信息追加在列表末尾。

如何获取我的 User ID？

登录牛客网后，点击页面右上角的头像，进入“我的主页”。此时，浏览器地址栏中的 URL 末尾的一串数字，就是您的 User ID。例如，https://www.nowcoder.com/users/12345678，那么 12345678 就是您的 User ID。

如何查询刷题记录？不同颜色代表什么？

在 “Problems” 标签页，于 “User ID” 输入框中填入您的牛客网用户ID，然后点击 “Search”。题目列表的状态便会高亮更新。绿色背景代表您已解决的题目，红色背景代表您尚未解决但对比用户已解决的题目。此外，当您解决了某场比赛中的所有题目后，该比赛的名称也会以绿色背景高亮显示，方便您快速识别已完全征服的比赛。
