常见问题解答

🎯 这个工具是做什么的？

这是一个专为算法学习者打造的牛客网刷题进度追踪器！无论你是算法竞赛新手、准备校招的同学，还是想要系统性学习算法的朋友，这个工具都能帮你：
✨ 每日打卡：每天一道精选题目，养成刷题好习惯
📊 进度追踪：一目了然看到自己的刷题进度和薄弱环节
🏆 排行榜：与全站用户对比，激发学习动力
🎯 精准定位：快速找到自己没做过的题目，高效刷题

📚 三个板块分别适合谁？

🏁 竞赛板块： 适合想要提升算法竞赛能力的同学！包含牛客周赛、小白月赛、练习赛、挑战赛、XCPC区域赛等各类竞赛题目，帮你锻炼代码思维和解题速度。

💼 校招板块： 准备技术面试的同学们看这里！汇集了美团、字节、阿里等大厂的校招真题，是面试准备的宝库。

📖 练习板块： 算法入门者的最佳选择！从"新手入门130"开始，循序渐进学习算法和数据结构，还有面试高频题等你来挑战。

🏆 排行榜是怎么排名的？

排行榜展示的是全站练习题目通过总数最多的用户！
📈 数据实时更新，确保排名准确性
🔍 支持搜索自己的排名位置
📊 如果您的排名在1000名以后，系统会特别显示您的排名信息
💡 小贴士：排行榜是激励自己进步的好工具，但不要过分焦虑，专注于自己的成长最重要！

🆔 如何找到我的 User ID？

超简单3步法：
1️⃣ 登录 牛客网
2️⃣ 点击页面右上角的头像
3️⃣ 进入"我的主页"，地址栏末尾的数字就是您的 User ID

示例： 如果地址是 https://www.nowcoder.com/users/12345678，那么您的 User ID 就是 12345678

💡 小贴士： 同样的方法也可以获取别人的 User ID 哦！只要访问他们的牛客主页，地址栏末尾的数字就是他们的 User ID（笑）

🎨 颜色代表什么意思？

我们用了直观的颜色系统帮您快速识别：
🟢 绿色背景：您已经通过的题目（成就感满满！）
🟠 橙色背景：您还没做但对比用户已通过的题目（学习目标！）
⚪ 白色背景：双方都没通过的题目（一起努力！）

特别提示： 当您完成某场比赛的所有题目时，比赛名称也会变成绿色，让您一眼看出自己的"完全征服"成就！

📅 每日打卡功能怎么用？

每日打卡是养成刷题习惯的绝佳方式！
🌅 每日一题：每天精选一道优质题目，难度适中
✅ 打卡记录：完成题目后点击打卡，记录你的坚持
📊 打卡日历：可视化展示你的打卡历史，满满的成就感
🎯 坚持奖励：连续打卡会显示在日历上，激励你继续前行

🚀 新手如何开始？

推荐学习路径：

🎯 算法竞赛方向（ACM/ICPC）：
1️⃣ 从"练习"板块开始：选择"新手入门130"，掌握基础算法
2️⃣ 参与每日打卡：每天完成一道题目，培养解题手感
3️⃣ 挑战竞赛题目：在"竞赛"板块练习牛客周赛、小白月赛等
4️⃣ 逐步进阶：掌握基础后尝试"算法进阶"、"算法登峰"
5️⃣ 补题训练：补练习赛和挑战赛的题，以及XCPC区域赛真题

💼 求职面试方向：
1️⃣ 从"练习"板块开始：选择"新手入门130"，打好基础
2️⃣ 参与每日打卡：每天完成一道题目，保持手感
3️⃣ 校招真题训练：在"校招"板块练习大厂真题
4️⃣ 面试高频题：重点学习"面试高频题目"和"面试top101"
5️⃣ 模拟面试：定期回顾错题，查漏补缺

记住：学习算法是一个循序渐进的过程，不要急于求成，坚持就是胜利！💪
